C++基础入门

数组

数组就是一个集合，里面存放着相同类型的数据元素

  • 数组的每个数据元素都是相同的数据类型
  • 数组是由连续的内存位置组成的

一维数组

  • 一维数组定义方式

    • 数据类型 数组名[数组长度];
    • 数据类型 数组名[数组长度] = {值1, 值2 ...};
    • 数据类型 数组名[] = {值1, 值2 ...};
      数组名命名规范与变量名一致，不要和变量重名，数组下标从0开始索引

  • 一维数组名称的用途

    • 统计整个数组在内存中的长度sizeof(arr)
    • 可以获取数组在内存中的首地址cout << arr;
      数组名是一个常量

  • #define宏常量:#define 常量名 常量值定义在上方
  • const修饰:const 数据类型 常量名 = 常量值;

标识符命名规则

  • 标识符不能是关键字
  • 标识符只能含有字母(区分大小)、数字和下划线
  • 首字符不能为数字
  • 尽量做到见名知意

数据类型

用于给变量分配合适的内存空间

整型

数据类型 占用空间 取值范围
short(短整型) 2字节 (-2^15 ~ 2^15-1)
int(整型) 4字节 (-2^31 ~ 2^31-1)
long(长整型) Windows为4字节,Linux为4字节(32位),8字节(64位) (-2^31 ~ 2^31-1)
long long(长长整型) 8字节 (-2^63 ~ 2^63-1)

占用空间大小可以用sizeof(int)或者sizeof(a)形式输出

实型(浮点型)

数据类型 占用空间 有效数字范围
单精度float 4字节 7位有效数字
双精度double 8字节 15~16位有效数字

有效数字：不止小数点后，小数点前面的也算位数；默认情况下输出有效数字位数为6位
定义float类型时最好末尾加f，因为小数默认是double类型，定义的时候会多一步转化

字符型

字符型变量用于显示单个字符，用单引号括起来
如char ch = 'a';，单引号内只能有一个字符，不能是字符串
字符型变量只占有1字节
字符型变量本质存储的不是字符，而是将对应的ASCII编码放入存储单元

ASCII码组成

  • ASCII非打印控制字符：0-31和127用于控制打印机等外围设备
  • ASCII打印字符：32-126分配给了能在键盘上找到的字符
    32-space，65-A，97-a

转义字符

作用：用于表示一些不能显示的ASCII字符，常用的有

转义字符 含义 ASCII码值(十进制)
\b 退格(BS),将当前位置移到前一列 008
\n 换行(LF),将当前位置移到下一行开头 010
\r 回车(CR),将当前位置移到本行开头 013
\t 水平制表(HT),跳到下一个TAB位置 009
\ \ 代表一个反斜线符号”\“ 092

字符串型

  • 延用C风格的字符串：char str1[] = "abcd";
  • C++风格的字符串：string str2 = "abcd"
  • 输出直接为cout << strn;

布尔类型 bool

所占内存为1字节,非0即为真。
true-真(本质为1)，false-假(本质为0)。

数据的输入

cin >> 变量;

运算符

作用：用于执行代码的运算

算术运算符(用于处理四则运算)

正号(+)、负号(-)、加(+)、减(-)、乘(*)、除(/)、取模或取余(%)、前置或后置递增(++)、前置或后置递减(–)
除法和取模运行算除数不能为0，两个小数不可以进行取模运算

赋值运算符(将表达式的值赋给变量)

赋值(=)、加等于(+=)、减等于(-=)、乘等于(*=)、除等于(/=)、模等于(%=)

比较运算符(用于表达式的比较并返回一个真值或假值)

相等于(==)、不等于(!=)、小于(<)、大于(>)、小于等于(<=)、大于等于(>+)

逻辑运算符(根据表达式的值返回真值或假值)

非(!)、与(&&)、或(||)

程序流程结构

三种程序运行结构：顺序结构、选择结构、循环结构

选择结构

if语句

作用：执行满足条件的语句

  • 单行格式if语句if(条件){条件满足执行的语句}
  • 多行格式if语句if(条件){条件满足执行的语句}else{条件不满足执行的语句}
  • 多条件的if语句if(条件1){条件1满足执行的语句}else if(条件2){条件2满足执行的语句}...else{都不满足执行的语句}
  • 嵌套if语句

三目运算符

如果为真执行2，否则3：表达式1 ？ 表达式2 ： 表达式3
比较三个数大小max = a>b ? ( a>c ? a : c ) : ( b>c ? b : c )

switch语句

执行多条件分支语句

switch的优点在于结构清晰，执行效率高
case语句中没有break的话程序会一直向下执行。
判断的时候只能是整型或字符型，不能判断区间，约束性较高。

循环结构

作用：满足循环条件，执行循环语句。

while循环语句

只要循环条件为真就执行循环语句while(循环条件){ 循环语句 }
循环一定要避免死循环的出现

do…while循环语句

先执行一次循环语句再判断循环条件do{ 循环语句 } while(循环条件)

for循环语句

满足条件执行循环语句：for(起始表达式; 条件表达式; 末尾循环体){ 循环语句; }
for循环结构清晰，比较常用

嵌套循环

打印乘法口诀表例题

跳转语句

break语句

作用：用于跳出选择结构或循环结构

  • 出现在switch条件语句中，作用是终止case并跳出switch
  • 出现在循环语句中，作用是跳出当前的循环语句
  • 出现在嵌套循环中，跳出最近的内层循环语句

continue语句

作用：在循环语句中，跳过本次循环中余下尚未执行的语句，继续执行下一次循环

goto语句

作用：若标记名称存在，执行到goto语句时，会无条件跳转到标记位置

不建议在程序中使用goto语句，以免造成程序流程的混乱。
