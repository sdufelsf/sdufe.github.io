原码 反码 补码

  1. 二进制最高位是符号位，0表示正数，1表示负数
  2. 整数的原码、反码和补码都一样
  3. 负数的反码 = 原码符号位不变，其他位取反
  4. 负数的补码 = 反码 + 1
  5. 0的反码和补码都是0
  6. java所有的数都是有符号的
  7. 计算机运算时是以补码的方式运算的
  8. 运算结果看的是原码

位运算符 (补码运算)

  1. 按位与&：两位全1则为1，否则为0
  2. 按位或|：两位有一个1则为1，否则为0
  3. 按位异或^：两位不同则为1，否则为0
  4. 按位取反~
  5. 算术右移>>：符号位不变，低位溢出并用符号位补溢出的高位
  6. 算术左移<<：符号不变，低位补0
  7. 逻辑右移也叫无符号右移>>>：低位溢出，高位补0

分支控制if-else语句

if括号里如果是关系运算符的话，就会判断整个表达式是true还是false，但是如果括号里是一个赋值语句的话，就比如说a=4;那么if(a=4)就等价于if(a)，这个a就是赋完值的a。但是在C/C++里，只有0是假，其他都为真，Java里只有true为真，java的boolean类型只有true和false，没有其他结果

switch语句

  • switch(表达式)中表达式的返回值必须是byte,short,int,char,enum[枚举],String
  • casez字句中的值必须是常量或常量表达式，不能是变量

break语句

continue和标签的用法同break

数组

boolean的初始值为false
基本数据类型之间的赋值方式是值传递(值拷贝)，互不影响
数组在默认情况下进行的是引用传递(地址拷贝)，赋的值是地址
