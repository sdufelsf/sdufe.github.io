Java的重要特点

1.Java语言是面向对象的(oop)
2.Java语言是健壮的。Java的强类型机制(明确的数据类型)、异常处理、垃圾的自动收集等是Java程序健壮性的重要保证
3.Java语言具有跨平台性(靠JVM实现)
4.Java语言是解释型的，解释型语言编译后的代码需要解释器才能执行，编译性语言编译后可以直接运行(C/C++)

环境变量

配置Path的作用是为了在dos的任意目录可以使用java和javac
配置用户变量仅对单一用户生效，系统变量适用于所有用户

JDK

JDK = JRE + java的开发工具(java,javac,javadoc,javap等)
JRE = JVM + Java SE标准类库(java核心类库)
.java文件通过javac生成.class文件，运行.class文件只需要JRE

注意事项和细节

Java应用程序的执行入口main()方法格式
public static void main(String[] args){...}
一个源文件最多只有一个public类，其他类不限
编译后每一个类都对应一个字节码文件
如果源文件包含了一个public类，文件名必须按该类命名
main方法可以写在非public类然后指定运行非public类

数据类型

基本数据类型

  • 数值型

    1. byte[1], short[2], int[4], long[8]
      整型常量默认int型，声明long型要加l或者L
    2. float[4], double[8]
      浮点数在机器中的存放形式,浮点数=符号位+指数位+尾数位
      0.125 等价于 .125
      浮点型常量默认double型,声明float型要加f或者F
      8.1/3 结果不是2.7，而是一个接近2.7的小数
      要比较两数的话可以使用差值的绝对值在某精度范围内判断

  • 字符型 char[2]

  • 布尔型 boolean[1]
    布尔类型的值只有false和true

常用编码

  1. ASCII码：美国指定的一套字符编码(1字节)，规定了英语字符和二进制之间的关系，最前面一位统一规定0，只用后七位表示了128个字符
  2. Unicode：一种编码包含世界上所有的符号，每个符号都有独一无二的编码，所以不会乱码，一个英文字母和汉字都是2字节，浪费空间
  3. UTF-8：是互联网上使用最广的一种Unicode实现方式，可以使用1-6个字节表示一个符号，根据不同符号变化字节长度，字母占1个字节，汉字占3个字节
  4. gbk：字母1字节，汉字2字节，表示的汉字比gbk2312多
  5. big5：表示繁体中文

算术运算符

取模的实质即为：a % b = a - a / b * b
如：-10%3= -10 - (-10)/3 *3 = -10+9 = -1
如果a或b有小数a/b*b的每个数要强制类型转换为int
面试题1

  1. 一个临时变量保存原来的i temp = i;
  2. 执行自增操作 i = i + 1;
  3. 把临时变量赋给i i = temp;
    所以答案为1

面试题2

  1. 执行自增操作 i = i + 1;
  2. 一个临时变量保存i temp = i;
  3. 把临时变量赋给i i = temp;
    所以答案为2

逻辑运算符

  • &&(短路与)和&(逻辑与)的判断条件相同，但是短路与高效，第一个条件false后不再判断，逻辑与全都会判断
  • ||(短路或)和|(逻辑或)同理，基本使用短路
    逻辑运算一定要注意赋值号和相等于！！！

标识符

Java标识符组成有字母、数字、_和$

  1. 包名：多单词组成时所有字母小写，如aaa.bbb.ccc
  2. 类名、接口名：多单词组成所有首字母大写，如XxxYyyZzz
  3. 变量名、方法名：多单词组合第一个单词首字母小写，，第二个单词开始首字母大写，如xxxYyyZzz
  4. 常量名：所有字母大写，多单词时下划线连接，如XXX_YYY_ZZZ
