Math，Arrays，System，BigInteger，BigDecimal，日期类的用法

Math类

包含用于执行基本数学运算的方法，如初等指数、对数、平方根和三角函数

常用方法

Math的常用方法均为static静态方法

  1. abs求绝对值

  2. pow求幂Math.pow(2, 4);表示2的4次方

  3. ceil向上取整，返回值为double类型

  4. floor向下取整

  5. round四舍五入

  6. sqrt 求开方

  7. random返回0-1之间的随机小数，左闭右开
    random返回a-b的随机整数x = (int)(a + Math.random()* (b-a+1));

  8. max和min

Arrays类

Arrays包含一系列静态方法，用于管理或操作数组(比如排序和搜索)

常用方法

  1. toString 返回数组的字符串形式

  2. sort

    1. 默认从小到大排序

    2. 调用定制排序传入两个参数为数组和实现Comparator接口的匿名内部类，方法返回值正负影响排序

  3. binarySearch 对排好序的数组进行二分查找，有返回index，没有返回- (应该存在的位置 + 1)

  4. copyOf(arr, 个数) 如果长度超过数组则多的为null，如果长度小于0抛出NegativeArraySizeException

  5. fill(num, 10) 使用数字10将num数组的元素全部替换掉

  6. equals 判断两个数组的元素是否完全相等

  7. asList 可以将一组值转换为list

System类

常用方法有：

  1. exit 退出当前程序，后面跟0表示正常退出

  2. arraycopy 复制数组元素，Arrays.copyOf的底层实现，比较适合底层调用

  3. currentTimeMillens 返回当前时间距离1970-1-1的毫秒数

  4. gc 运行垃圾回收机制

BigInteger类和BigDecimal类

  1. 使用字符串构造对象，使用相应的add，subtract，multiply，divide方法

  2. BigDecimal的decide可能会因为除不尽抛出ArithmeticException，解决方法为做除法时指定精度，如System.out.println(bigDec.divide(bigDec2, BigDecimal.ROUND_CEILING));会保留分子的精度

第三代日期类(JDK8加入)

JDK1.0包含了java.util.Date类，但是大多数方法在JDK1.1引入Calendar类后被弃用，Calendar存在的缺点：

  1. 可变性：日期和时间这样的类应该是不可变的，在多线程环境下需要额外的同步操作才能保证线程安全

  2. 偏移性：Date中的年份编号是基于1900年开始的，月份都是从0开始的

  3. 格式化：Calendar不能格式化

  4. 线程不安全；不能处理闰秒等(每隔两天多出1s)
