前端Web开发

HTML & CSS

HTML结构标签

特点

  1. HTML标签不区分大小写
  2. HTML标签属性值单双引号都可以
  3. HTML语法比较松散

标签

  1. 标题标签: <h1>...</h1> h1 -> h6重要程度依次降低
  2. 水平线标签 <hr>
  3. 图片标签 <img src='...' width='...' height='...'> 长宽只写一个会等比例缩放
  4. <span> 标签是无语义的布局标签,一行可以使用多次,宽高由内容撑开
  5. 视频标签 <video src="..." controls width="900px"></video> controls为显示播放控件, 当HTML中的标签名和属性值一样就可以省略
  6. 音频标签 <audio src="..." controls></audio>
  7. 段落标签 <p>
  8. 文本加粗标签 <b> 或者 <strong>

CSS引入方式

  1. 行内样式: 写在标签style属性里面(不推荐)
  2. 内嵌样式: 写在style标签中(可以写在页面的任意位置,但是通常约定写在head标签中)
  3. 外联样式: 写在一个单独的 .css 文件中(需要通过 <link href="..."> 标签在网页中引入)

颜色表示形式

  1. 关键字,预定义的颜色名,如red, green, blue等
  2. rgb表示法,红绿蓝三原色,每项取值范围0~255, 如rgb(0,0,0), rgb(255, 255, 255)等
  3. 十六进制表示法用#开头将数字转换为十六进制, 如#000000简写#000,两位相同都可以简写

CSS选择器

用来选取需要设置样式的元素, id选择器的优先级大于类选择器大于元素选择器

  1. 元素选择器 元素名称 {color: red;}
  2. id选择器 #id属性值 {color: red;} id不可重复
  3. 类选择器 .class {color: red;} 类名可以重复

超链接

超链接标签 <a href="..." target="...">xxx</a> target有_self和_blank 值来表示在哪打开链接,默认选择self当前页面加载,修改a的style可以将超链接的颜色和下划线去除
