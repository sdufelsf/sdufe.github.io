神经网络基础

二分分类

二分类问题中的目标就是习得一个分类器，它以图片的特征向量(将图片的红绿蓝像素值数量n=n~x~作为维度)作为输入，可能会用到X.shape(n, m)，然后预测输出y的结果为1还是0，可能会用到Y.shape(1, m)

Logistic回归

给定输入特征向量X后，要预测结果 y^\widehat{y}y​ ，就要通过改变特征向量中各值所占的权重 (w即表示特征权重，维度和特征向量相同) 来使y的值接近1，所以使 用y^=wTx+b\widehat{y} = w^Tx + by​=wTx+b​进行预测，b为一个表示偏差的实数（公式具体解释可以参考超平面、Wx=b或Wx+b=0的几何意义）但是这个线性函数的问题在于预测结果可能为负值或者远大于1的数值，而我们进行二元分类时数据只有在0到1之间才有意义，因此就使用了非线性的sigmoid函数使结果介于0-1之间

S(x)=11+e−x\mathrm{S}(\mathrm{x})=\frac1{1+\mathrm{e}^{-\mathrm{x}}} S(x)=1+e−x1​

Logistic回归损失函数

损失函数是用来衡量预测输出值与实际值情况的，在逻辑回归中用到的损失函数为

L(y^,y)=−ylog(y^)−(1−y)log(1−y^)L(\widehat{y}, y) = -ylog(\widehat{y}) - (1 - y)log(1-\widehat{y}) L(y​,y)=−ylog(y​)−(1−y)log(1−y​)

使用该函数就可以达到y取1或0时 y^\widehat{y}y​​ 的值无限接近1或0

损失函数是在单个训练样本中定义的，它衡量的是算法在单个训练样本中表现如何，为了衡量算法在全部训练样本上的表现如何，我们需要定义一个算法的代价函数，算法的代价函数就是对 m个样本的损失函数求和然后除以 m :

J(w,b)=1m∑i=1mL(y^(i),y(i))=1m∑i=1m(−y(i)log⁡y^(i)−(1−y(i))log⁡(1−y^(i)))\mathrm{J(w,b)=\frac{1}{m}\sum_{i=1}^m L(\hat{y}^{(i)},y^{(i)})=\frac{1}{m}\sum_{i=1}^m(-y^{(i)}\log\hat{y}^{(i)}-(1-y^{(i)})\log(1-\hat{y}^{(i)}))} J(w,b)=m1​i=1∑m​L(y^​(i),y(i))=m1​i=1∑m​(−y(i)logy^​(i)−(1−y(i))log(1−y^​(i)))

所以在训练逻辑回归模型时候，我们需要找到合适的 w 和 b ，来让代价函数 J 的总代价降到最低

梯度下降法

在测试集上，要通过最小化代价函数(成本函数)来训练参数w和b，逻辑回归的代价函数是一个凸函数，所以初始化w和b后(凸函数无论在哪初始化应该都能达到同一点或大致相同的点)，通过朝最抖的坡度不断迭代最终走到全局最优解或者接近全局最优解的地方，逻辑回归的代价函数两个参数的修正量为：

w:=w−α∂J(w,b)∂wb:=b−α∂J(w,b)∂b\begin{gathered} w:=w-\alpha\frac{\partial J(w,b)}{\partial w} \\ b:=b-\alpha\frac{\partial J(w,b)}{\partial b} \end{gathered} w:=w−α∂w∂J(w,b)​b:=b−α∂b∂J(w,b)​​

导数为负时会不断向正方向移动，导数为正时会向负方向移动，α\alphaα​为学习率，因此要实现逻辑回归的梯度下降算法首先就要求出损失函数 J 对 w 和 b 的偏导，链式法则先求出 J 对 z 的偏导为

这种计算的缺点是在逻辑回归时你需要编写两个for循环，第一个for循环遍历 m 个训练样本，第二个for循环是遍历每个样本所有特征的for循环，但是当数据很大时在代码中显式的使用for循环是很低效的，因此要用到向量化

向量化

大写字母代表向量，小写字母代表元素

向量化是非常基础的去除代码中for循环的艺术，比如运算z=wTx+bz = w^Tx + bz=wTx+b（一百万数据量）使用z = np.dot(w, x) + b比使用for循环实现快了三百多倍。无论什么时候，都要尽量使用内置函数而避免for循环

比如m个样本实例进行逻辑回归的梯度下降时，可以使用dw = np.zeros(n(x),1)来定义一个dw向量，使用 xnx_nxn​ 行的一维向量代替样本特征，只需要dw += x(i)dz(i)就可以去除内部for循环，通过定义矩阵X作为训练输入，大小为( nxn_xnx​​ ，m)，然后只需要Z = np.dot(w.T,X) + b就可以将外层循环也消除掉，因此我们只要利用下面公式即可完成前向和后向传播，实现对所有训练样本进行预测和求导

Z=wTX+b=np.dot(w.T,x)+bA=σ(Z)dZ=A−Ydw=1m∗X∗dzTdb=1m ∗np.sum(dZ)\mathrm{Z=w^T X+b=np.dot(w.T,x)+b}\\ \mathrm{A}=\sigma(\mathrm{Z})\\ \mathrm{dZ}=\mathrm{A}-\mathrm{Y}\\ \mathrm{dw}=\frac1{\mathrm{m}}*\mathrm{X}*\mathrm{dz}^\mathrm{T}\\ \mathrm{db=\frac1m~*np.sum(dZ)} Z=wTX+b=np.dot(w.T,x)+bA=σ(Z)dZ=A−Ydw=m1​∗X∗dzTdb=m1​ ∗np.sum(dZ)

然后再利用 w 和 b 的公式梯度下降更新参数，以此实现了不使用for循环通过一次迭代实现一次梯度下降

浅层神经网络

神经网络表示

输入的特征堆叠在一起叫做神经网络的输入层，中间有隐藏层，最后的结点为产生预测值的输出层；隐藏层的含义是当我们使用监督学习训练一个神经网络时，训练集包括了输入 x 和目标输出 y，但是中间节点也就是隐藏层的准确值我们是看不见的

在神经网络中，我们将使用带放括号上标的形式来明确地指出值来自于哪一层，当我们计算网络的层数时，输入层是不算入总层数内的（可以记作第0层）

神经网络的输出

逻辑回归的计算有两个步骤，首先你按步骤计算出 z ，然后在第二步中你以sigmoid函数为激活函数计算z求出a，一个神经网络就是做了好多次这种重复计算，也就是一个神经网络就是多个逻辑回归的计算单元组成的，进行向量化的表示就是

z[n]=w[n]x+b[n]a[n]=σ(z[n])\mathrm{z^{[n]}=w^{[n]}x+b^{[n]}}\\ \mathrm{a^{[n]}=\sigma(z^{[n]})} z[n]=w[n]x+b[n]a[n]=σ(z[n])

而每一个隐藏层的输入就是上一个隐藏层的输出，就比如

Given  input  x:z[1]=W[1]x+b[1]a[1]=σ(z[1])z[2]=W[2]a[1]+b[2]a[2]=σ(z[2])Given{\,\,}input{\,\,}x:\\ z^{[1]} = W^{[1]}x + b^{[1]}\\ a^{[1]} = \sigma(z^{[1]})\\ z^{[2]} = W^{[2]}a^{[1]} + b^{[2]}\\ a^{[2]} = \sigma(z^{[2]}) Giveninputx:z[1]=W[1]x+b[1]a[1]=σ(z[1])z[2]=W[2]a[1]+b[2]a[2]=σ(z[2])

每个样本都需要进行如上的操作，从水平上看，矩阵 A 代表了各个训练样本。从竖直上看，矩阵 A 的不同的索引对应于不同的隐藏单元。对于矩阵 Z ，X情况也类似，水平方向上，对应于不同的训练样本；竖直方向上，对应不同的输入特征，而这就是神经网络输入层中各个节点。因此对应的向量化实现的效果也就是

z[1]=W[1]X+b[1]A[1]=σ(z[1])z[2]=W[2]A[1]+b[2]A[2]=σ(z[2])\mathrm{z}^{[1]}=\mathrm{W}^{[1]}\mathrm{X}+\mathrm{b}^{[1]}\\ \mathrm{A}^{[1]}=\mathrm{\sigma}(\mathrm{z}^{[1]})\\ \mathrm{z}^{[2]}=\mathrm{W}^{[2]}\mathrm{A}^{[1]}+\mathrm{b}^{[2]}\\ \mathrm{A}^{[2]}=\sigma(\mathrm{z}^{[2]}) z[1]=W[1]X+b[1]A[1]=σ(z[1])z[2]=W[2]A[1]+b[2]A[2]=σ(z[2])

激活函数

在前面的学习中我们一直在用的激活函数只有sigmoid函数，深度网络有很多很多隐藏层，如果你使用线性激活函数或者没有使用激活函数，那么无论你的神经网络有多少层一直在做的只是计算线性函数，而两个线性函数的组合本身就是线性函数，所以不如直接去掉全部隐藏层

tanh(双曲正切函数)是sigmoid的向下平移和伸缩后的结果。对它进行了变形后，穿过了(0, 0)点，并且值域介于+1和-1之间，结果表明，如果在隐藏层上使用tanh函数效果总是优于sigmoid函数。因为函数值域在-1和+1的激活函数，其均值是更接近零均值的。在训练一个算法模型时，如果使用tanh函数代替sigmoid函数中心化数据，使得数据的平均值更接近0而不是0.5

a=tanh⁡(z)=ez−e−zez+e−z\mathrm{a=\tanh(z)=\frac{e^z-e^{-z}}{e^z+e^{-z}}} a=tanh(z)=ez+e−zez−e−z​

但是二分类问题中想让预测值结果在0和1之间所以需要使用sigmoid函数

sigmoid函数和tanh函数两者共同的缺点是，在 z 特别大或者特别小的情况下，导数的梯度或者函数的斜率会变得特别小，最后就会接近于0，降低梯度下降的速度。所以使用了修正线性单元的函数ReLu

a=max(0,z)a = max(0,z) a=max(0,z)

从实际上来说，当使用 z 的导数时，z = 0的导数是没有定义的。但是当编程实现的时候，z 的取值刚好等于接近于0的数值，这个值相当小，所以在实践中，不需要担心这个值，z等于0的时候，假设一个导数是1或者0效果都可以；也有另一个版本的Relu被称为Leaky Relu，当 z 是负值时函数的值不是等于0，而是轻微的倾斜，也就是 a=max(αz,z)a = max(\alpha z,z)a=max(αz,z) 的效果，α\alphaα 的取值一般为0.01，这个函数的效果通常比Relu更好，但是也存在效果不明显的情况

随机初始化

对于逻辑回归，把权重初始化为0当然也是可以的；但是对于一个神经网络，如果你把权重或者参数都初始化为0，那么梯度下降将不会起作用，简单地说就是会导致每个激活单元的函数数据完全相同，这种情况下超过1个隐含单元也没什么意义，因为他们计算同样的东西，具体解释可以参考一下这篇文章：谈谈神经网络权重为什么不能初始化为0；因此我们可以生成高斯分布的随机初始化

W[1]=np.random.randn(2,2)∗0.01,b[1]=np.zeros((2,1))W[2]=np.random.randn(2,2)∗0.01,b[2]=0\mathrm{W}^{[1]}=\mathrm{np.random.randn}(2,2)*0.01,\mathrm{b}^{[1]}=\mathrm{np.zeros}((2,1))\\\mathrm{W}^{[2]}=\mathrm{np.random.randn}(2,2)*0.01,\mathrm{b}^{[2]}=0 W[1]=np.random.randn(2,2)∗0.01,b[1]=np.zeros((2,1))W[2]=np.random.randn(2,2)∗0.01,b[2]=0

这个常数从哪里来，为什么是0.01，而不是100或者1000。我们通常倾向于初始化为很小的随机数。因为如果你用tanh或者sigmoid激活函数，或者说只在输出层有一个Sigmoid，如果（数值）波动太大，当你计算激活值时如果 W 很大， z 就会很大或者很小，因此这种情况下你很可能停在tanh/sigmoid函数的平坦的地方(见图3.8.2)，这些地方梯度很小也就意味着梯度下降会很慢，因此学习也就很慢。
